# Library

This directory stores original reference materials and links.

## Layout

- `PDF/essay/`: research papers and essays.
- `PDF/tech_manual/`: NVIDIA whitepapers, manuals, tutorials, and technical guides.
- `repo/`: links to code repositories. Do not copy or clone repositories here unless the project policy changes.

## Rule

Keep original materials here. Derived notes belong in `00-reference_essay/`, `01_hardware/`, `02_tools/`, or `03_operators/`.
