# Design Specs

This directory stores design documents for the model and experiment methodology.

Use it for:

- modeling assumptions,
- calibration strategy,
- power measurement environment requirements,
- reasons for changing the model,
- design tradeoffs that should persist across experiments.

Do not store per-run results here. Per-run results belong in `experiments/` and summarized in `QUALITY.md`.

## Current Specs

- `power_measurement_environment.md`: required environment for valid H800 power measurement.
